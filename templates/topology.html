Topology REST API

GET
/controller/nb/v2/topology/{containerName}
Retrieve the Topology

Retrieve the Topology for a certain Container

Response Codes
HTTP Status Code Description
404 Not Found The Container Name was not found
Request parameters
name description type default
containerName The container for which we want to retrieve the topology (Eg. 'default') path
_q (no documentation provided) query
Response parameters
name description type default
topology A list of edgeProperties element
edgeProperties One edge and its properties element
edge A directed edge connecting headNodeConnector and tailNodeConnector element
properties The properties of an edge element
tailNodeConnector A nodeConnector which means a generic network element attachment points, attached to one node element
headNodeConnector A nodeConnector which means a generic network element attachment points, attached to one node element
node A generic network element in multiple SDNs technologies. A Node is identified by the pair (id, type) element
id Element that constitutes the NodeConnector element
type Element that constitutes the NodeConnector element
state State of an edge, "1" means up and "0" means down property
config Admin config status of an edge, "1" means up and "0" means down property
name Name of an edge property
timeStamp The time when something happened on edge property

http://localhost:8080/controller/nb/v2/topology/default

PUT
/controller/nb/v2/topology/{containerName}/userLink/{name}
Add an User Link

Add an User Link for a certain Container

Response Codes
HTTP Status Code Description
201 Created User Link added successfully
404 Not Found The Container Name was not found
409 Conflict Failed to add User Link due to Conflicting Name
500 Internal Server Error Failed to add User Link. Failure Reason included in HTTP Error response
503 Service Unavailable One or more of Controller services are unavailable
Request parameters
name description type default
containerName Name of the Container (Eg. 'default') path
name Name of the user link path
topologyUserLinkConfig The configuration of a user link element
status The status of a user link which wanted to be set element
srcNodeConnector The configuration of a user link element
dstNodeConnector The configuration of a user link element
Request example
Request URL:
http://localhost:8080/controller/nb/v2/topology/default/userLink/link1
Request body in XML:
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<topologyUserLinkConfig>
  <status>Success</status>
  <name>link1</name>
  <srcNodeConnector>OF|2@OF|00:00:00:00:00:00:00:02</srcNodeConnector>
  <dstNodeConnector>OF|2@OF|00:00:00:00:00:00:00:51</dstNodeConnector>
</topologyUserLinkConfig>
Request body in JSON:
{
  "status":"Success",
  "name":"link1",
  "srcNodeConnector":"OF|2@OF|00:00:00:00:00:00:00:02",
  "dstNodeConnector":"OF|2@OF|00:00:00:00:00:00:00:51"
}
DELETE
/controller/nb/v2/topology/{containerName}/userLink/{name}
Delete an User Link

Delete an User Link for a certain Container

Response Codes
HTTP Status Code Description
204 No Content User link removed successfully
404 Not Found The Container Name or Link Configuration Name was not found
503 Service Unavailable One or more of Controller services are unavailable
Request parameters
name description type default
containerName Name of the Container (Eg. 'default') path
name Name of the Link Configuration (Eg. 'config1') path
Request example
Request URL:
http://localhost:8080/controller/nb/v2/topology/default/userLink/config1
GET
/controller/nb/v2/topology/{containerName}/userLinks
Retrieve the user configured links

Retrieve the user configured links for a certain Container

Response Codes
HTTP Status Code Description
404 Not Found The Container Name was not found
Request parameters
name description type default
containerName The container for which we want to retrieve the user links (Eg. 'default') path
_q (no documentation provided) query
Response parameters
name description type default
list A list of userLinks element
userLinks A list of userLinks element
status The status of a userLink property
name The name of a userLink element
srcNodeConnector The source NodeConnector of a userLink element
dstNodeConnector The destination NodeConnector of a userLink element
Request example
Request URL:
http://localhost:8080/controller/nb/v2/topology/default/userLinks
Response example
Response body in XML:
<?xml version="1.0" encoding="UTF-8" standalone="yes"?>
<list>
   <userLinks>
      <status>Success</status>
      <name>link1</name>

      <dstNodeConnector>OF|2@OF|00:00:00:00:00:00:00:51</dstNodeConnector>
   </userLinks>
</list>
Response body in JSON:
{
  "userLinks": [
    {
