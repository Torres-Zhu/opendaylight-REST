Host Tracker REST API

GET
/controller/nb/v2/hosttracker/{containerName}/address/{networkAddress}
                         Returns a host

                         Returns a host that matches the IP                               Address value passed as parameter

 
Response Codes
HTTP Status Code Description
200 OK Operation successful
400 Bad Request Invalid IP specified in networkAddress parameter
404 Not Found The containerName is not found
503 Service Unavailable One or more of Controller Services are unavailable
Request parameters
name description type default
containerName Name of the Container. The Container name for the base controller is "default" path
networkAddress IP Address being looked up path 
Request URL:
http://localhost:8080/controller/nb/v2/hosttracker/default/address/1.1.1.1
Response body in XML

<hostConfig>
 <dataLayerAddress>00:00:00:00:01:01</dataLayerAddress>
 <networkAddress>1.1.1.1</networkAddress>
 <nodeType>OF</nodeType>
 <nodeId>00:00:00:00:00:00:00:01</nodeId>
 <nodeConnectorType>OF</nodeConnectorType>
 <nodeConnectorId>9</nodeConnectorId>
 <vlan>0</vlan>
 <staticHost>false</staticHost>
</hostConfig>
Response body in JSON:

{
 "dataLayerAddress":"00:00:00:00:01:01",
 "nodeType":"OF",
 "nodeId":"00:00:00:00:00:00:00:01",
 "nodeConnectorType":"OF",
 "nodeConnectorId":"9",
 "vlan":"0",
 "staticHost":"false",
 "networkAddress":"1.1.1.1"
}
PUT
/controller/nb/v2/hosttracker/{containerName}/address/{networkAddress}
                         Add a Static Host configuration

                         Add a Static Host configuration. If a                            host by the given address already                               exists, this method will respond with                            a non-successful status response

 
Response Codes
HTTP Status Code Description
201 Created Static host created successfully
400 Bad Request Invalid parameters specified, see response body for details
404 Not Found The container or resource is not found
409 Conflict Resource conflict, see response body for details
503 Service Unavailable One or more of Controller services are unavailable
Request parameters
name description type default
containerName Name of the Container. The Container name for the base controller is "default" path
networkAddress Host IP Address path 
Request URL:
http://localhost:8080/controller/nb/v2/hosttracker/default/address/1.1.1.1
Request body in XML

 <dataLayerAddress>00:00:00:00:01:01</dataLayerAddress>

 <nodeConnectorType>OF</nodeConnectorType>

                         Delete a Static Host configuration

 
Response Codes
HTTP Status Code Description

404 Not Found The container or a specified resource was not found

503 Service Unavailable One or more of Controller service is unavailable
Request parameters

containerName Name of the Container. The Container name for the base controller is "default" path
networkAddress Host IP Address path 
Request URL:
http://localhost:8080/controller/nb/v2/hosttracker/default/address/1.1.1.1
